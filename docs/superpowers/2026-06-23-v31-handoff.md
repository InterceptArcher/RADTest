## How to test (reference)
Smoke a job + download/inspect the deck:
```
curl -X POST $BACKEND/profile-request -H 'Content-Type: application/json' \
  -d '{"company_name":"Microsoft","domain":"microsoft.com","requested_by":"x@y.z","salesperson_name":"Test"}'
# poll $BACKEND/job-status/{id}; deck URL is result.slideshow_url (…/decks/{id}.pptx)
# $BACKEND/debug-v31 reports flag/anthropic/last_v31_error
```
Inspect a deck by reading only the slides in `presentation.xml` sldIdLst order (orphaned parts otherwise inflate counts).
